# ConsertaFlix Academy - Landing Page

Landing page educacional para cursos de reparo celular da **ConsertaFlix Academy**.

## 🎨 Design e Estilo

- **Padrão:** Tech Industrial
- **Paleta de cores:** Azul marinho (#1e3a8a, #0f172a) + Laranja/Amarelo (#f59e0b)
- **Tipografia:** Inter (Google Fonts)
- **Stack:** HTML puro + Tailwind CSS via CDN

## 📚 O Curso

Curso online completo para aprender reparo de celulares com módulos práticos em:

- Esquemas Elétricos
- CPU Avançado
- Fonte Bancada
- Reballing
- Andamento de Reparo
- Medição de Capacitores
- Desconexão de CPU

### 📋 Conteúdo Programático

1. **Esquema Elétrico** - Análise de circuitos e diagnósticos precisos
2. **CPU Avançado** - Micro-soldagem e reballing avançado
3. **Fonte Bancada** - Testes de diagnóstico de componentes
4. **Reballing** - Técnicas avançadas de solda a quente

## 🚀 Deploy em GitHub Pages

### Opção 1: gh-pages (recomendado)

```bash
# 1️⃣ Adicionar remote origin do seu repositório
git remote add origin https://github.com/SEU-USUARIO/CONsertaflix-academy.github.io.git

# 2️⃣ Push para a branch main
git push -u origin main --tags
```

O site estará disponível em: `https://seu-usario.github.io/consertaflix-academy`

### Opção 2: Vercel/Netlify (mais rápido)

- **Vercel:** [vercel.com/new](https://vercel.com/new?repository-url=https://github.com/SEU-USUARIO/seu-repo)
- **Netlify:** [app.netlify.com/drop](https://app.netlify.com/drop)

Basta arrastar o repositório ou fazer o clone e conectar.

### Auto-deploy automático

O workflow `pages.yml` já está configurado para publicar automaticamente toda vez que você fizer um push na branch `main`.
